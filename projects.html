Nyayanidhi Platform

An AI-powered legal document processing platform for Indian law firms. I solo-built the entire backend using FastAPI, async SQLAlchemy, and PostgreSQL on GCP serverless (Cloud Run, Workflows, Tasks). Key achievements:

  • Document Ingestion: 7-phase pipeline processing 500+ PDFs in under 10 minutes with fan-in/fan-out orchestration.
  • Multi-Agent Drafting: Generates contracts from 19 variant templates using schema-driven LLM tool creation.
  • LLM Infrastructure: Prompt caching (Anthropic/Gemini), multi-modal support, provider migration, and optimized connection pools that cut p95 API latency by 40%.
  • Collaborative Editing: Y.js integration and Redis Pub/Sub for real-time collaboration.
