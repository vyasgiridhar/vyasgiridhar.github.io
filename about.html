Nyayanidhi · Building AI Platforms from Scratch

I am a Senior Backend Engineer at Nyayanidhi, a legal-tech startup building AI-powered document processing for Indian law firms. I architected and built the core platform end-to-end on GCP serverless using FastAPI, async SQLAlchemy, and PostgreSQL.

My core focus is on scalable, resilient data pipelines and multi-agent systems. I designed a 7-phase document ingest pipeline using Cloud Workflows that processes 500+ PDFs in under 10 minutes, with OCR, translation, and vectorization—each phase independently restartable via checkpoint-based resumption.

On the AI side, I built a multi-agent drafting system that generates contracts from 19 variant templates using schema-driven LLM tool creation. I manage the LLM infrastructure directly: prompt caching across Anthropic and Gemini, multi-modal content support, provider abstractions, and connection pool tuning that reduced p95 API latency by 40%.

LRZ · Supercomputing at Scale

At the Leibniz Supercomputing Centre (LRZ) in Munich, I worked on the infrastructure team managing HPC clusters with 1000+ compute nodes connected via InfiniBand. This was bare-metal systems engineering—no cloud abstractions.

I built automation tooling for patch workflows and monitoring, and profiled system performance at the hardware level using perf, Intel VTune, and Valgrind. I bridged the gap between cloud-native architecture and deep systems engineering (NUMA topology, cache hierarchies, MPI, OpenMP, SLURM).

Fiserv · Scale Under Pressure

During the COVID pandemic, I helped scale the Money Network infrastructure for the US Treasury's Economic Impact Program, delivering stimulus debit cards to 3 million Americans. Operating under strict government deadlines and zero tolerance for downtime, I rebuilt the microservices backend for the customer care system (Spring Boot, supporting 2000+ programs) and re-architected a disputes platform handling over 1 million daily transactions.
